# Personal Portfolio Website

## Description

This is a personal portfolio website developed using **HTML, CSS, and JavaScript**. It showcases my profile, skills, projects, and contact information. The website is responsive, user-friendly, and includes simple interactive features.

## Features

- Home section
- About Me section
- Skills section
- Projects section with GitHub links
- Contact form with basic validation
- Dark mode
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
